
  # About the project 📝  
  A backend system built with Laravel that fetches articles from selected data sources

  ## Get Started 🚀  
  Install Dependencies
  ~~~
  composer install 
  ~~~
  Start the artisan server
  ~~~
    php artisan serve
  ~~~
